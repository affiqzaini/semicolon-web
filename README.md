# nuxt-ts-vuetify

## Cloning Guides

```bash
# install dependencies
$ npm install

# create .env file

# check remote url
$ git remote -v

# remove git remote origin
$ git remove rm origin

# add new remote url
$ git remote add origin https://github.com/user/repo.git

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
